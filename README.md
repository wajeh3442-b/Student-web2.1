# List Example (HTML Assignment)

## 📌 Description

This project is a simple HTML webpage that demonstrates the difference between an unalphabetized list and an alphabetized list. It displays a set of names in their original order and then shows the same names sorted alphabetically for comparison.

## 🛠 Features

* Two sections:

  * Unalphabetized list
  * Alphabetized list
* Use of `<sub>` tag to show position numbers
* Clear comparison between unordered and ordered data
* Simple and clean structure

## 💻 Technologies Used

* HTML5

## 📂 File Structure

* `index.html` – Contains the complete code

## 🚀 How to Run

1. Save the file as `index.html`
2. Open it in any web browser
3. View both lists and compare the order

## 🎯 Purpose

This assignment helps in understanding:

* Basic HTML structure
* Displaying lists using text and line breaks
* Alphabetical sorting concept
* Use of subscript tags
